# Проект MathChecker

Api для проверки математической формулы

## Требования

- Docker
- Docker Compose

## Установка и запуск

1. **Клонирование репозитория**:
```bash
git clone https://github.com/SANJIKS/MathChecker
```

2. **Переход в директорию проекта**:
```bash
cd MathChecker
```

3. **Построение образов с помощью Docker Compose**:
```bash
docker-compose build
```

4. **Запуск сервисов**:
```bash
docker-compose up
```

Присутствуют тесты и журнал запросов.
